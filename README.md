# 💼 Portfólio Pessoal

Este projeto é o meu portfólio pessoal, desenvolvido com o objetivo de apresentar minha trajetória, formação, habilidades e projetos como Desenvolvedor Full Stack.

Além de servir como um cartão de visitas profissional, o portfólio reúne alguns dos projetos que desenvolvi durante meus estudos e demonstra meus conhecimentos em desenvolvimento web.

---

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Font Awesome

---

## ✨ Funcionalidades

* 📌 Seção "Sobre Mim"
* 🎓 Seção de Formação Acadêmica
* 📂 Exibição de projetos desenvolvidos
* 📧 Formulário de contato com validação em JavaScript
* ✔️ Validação de campos obrigatórios
* ✔️ Validação de e-mail
* ✔️ Mensagem de confirmação após envio
* 📱 Layout responsivo para diferentes dispositivos

## ▶️ Como executar

1. Clone este repositório:

```bash
git clone https://github.com/devkaikepereira/portfolio.git
```

2. Abra a pasta do projeto.

3. Execute o arquivo `index.html` diretamente no navegador ou utilize uma extensão como **Live Server** no Visual Studio Code.
